E-commerce diseñado en una arquitectura cliente-servidor.

Esta diseñado para la visualización y pedidos de productos de la marca Black button,
una marca joven y 100% Méxicana la cual necesita expandir sus estrategías para
darse a conocer así como mejorar sus formas de venta.

Este e-commerce se encarga de mostrar de una manera fácil los productos, pueden
ser agregados a carrito para su posterior pedido, el cual se realiza generando un
código de pedido y enviando junto con toda la información de los productos utilizando
la api de whatsapp para que el usuario pueda comunicarse con la creadora de los
productos.

Entorno producción

[Producción](https://blackbuttondev.fjdev.icu/)

El proyecto esta desplegado en un servidor VPS utilizando:

    Web Server: Ngix.
    Aplication Server: Gunicorn
    SSL
